# Hand Draw UI Kit

A full interactive demo of the Hand Draw Design System built as a landing page prototype.

## What's included

- **Navbar** — logo, links with draw-on underlines, CTA button
- **Hero** — marker-highlight headline, italic subtext, primary + ghost buttons, decorative arrow
- **Features grid** — three sticky-note cards with SVG frames, icons, stamp decal
- **Contact form** — grid/dotted paper surface, framed inputs, margin note annotation
- **Quote section** — SVG drawn quotation marks, Caveat display type, double underline author
- **Principles** — ruled paper background, numbered list in Caveat
- **Footer** — wavy SVG rule, drawn pen illustration

## How to use

Open `index.html` in a browser. All fonts load from Google Fonts CDN.

To use components in your own work:
- Copy the relevant `<section>` blocks
- Reference `../../colors_and_type.css` for tokens
- The SVG `#svg-defs` block (filters) must be present in the page
